
# MicGoFiber

Microservice boilerplate for Go Fiber project

## Run Locally

Clone the project

```bash
  git clone https://github.com/waynemamahit/micgofiber
```

Go to the project directory

```bash
  cd micgofiber
```

Install dependencies

```bash
  go mod tidy
```

Start the server

```bash
  go run main.go
```

Start the server with Air

```bash
  air
```
